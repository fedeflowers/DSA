## 1. Approach Explanation

The solution uses Common Table Expressions (CTEs) to break down the problem into manageable parts. The goal is to determine:

- The user who has rated the most movies.
- The movie that has the highest average rating from user ratings in February 2020.

### Steps:

1. **Compute the Most Active Rater (`res1`)**:
   - A CTE named `res1` is created, which retrieves the user's name who has given the most ratings.
   - It joins the `movierating` table with the `users` table based on `user_id`.
   - The results are grouped by `user_id` and `name`, counting the number of ratings for each user.
   - The result is ordered by the count of ratings in descending order, and by user names in ascending order.
   - Only the top user (most ratings) is selected using `LIMIT 1`.

2. **Compute the Highest Rated Movie (`res2`)**:
   - Another CTE, `res2`, calculates the average rating of movies rated in February 2020.
   - It joins the `movierating` table with the `movies` table based on `movie_id`.
   - The `WHERE` clause filters ratings to include only those created in February 2020.
   - Results are grouped by `movie_id`, and the average rating is calculated.
   - The result is ordered by average rating in descending order, and by movie titles in ascending order.
   - Similarly, only the top movie is selected using `LIMIT 1`.

3. **Combine Results**:
   - Finally, the results from both CTEs are selected and combined using `UNION ALL` to return both the highest rater and the highest rated movie.

### Time Complexity:

  - The join operation and the group by on `user_id` typically takes O(N), where N is the number of ratings.

WITH res1 AS (
    SELECT u.name AS results
    FROM movierating mr
    JOIN users u ON mr.user_id = u.user_id
    GROUP BY mr.user_id, u.name
    ORDER BY COUNT(*) DESC, u.name ASC
    LIMIT 1
),
res2 AS (
    SELECT m.title AS results
    FROM movierating mr
    JOIN movies m ON mr.movie_id = m.movie_id
    WHERE YEAR(mr.created_at) = 2020 AND MONTH(mr.created_at) = 2
    GROUP BY mr.movie_id, m.title
    ORDER BY AVG(mr.rating) DESC, m.title ASC
    LIMIT 1
)

SELECT * FROM res1
UNION ALL
SELECT * FROM res2

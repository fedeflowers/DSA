  - `first_y`: This CTE uses the `ROW_NUMBER()` window function to enumerate the years for each product, partitioned by `product_id` and ordered by `year` in ascending order. This allows us to assign a unique row number (`c`) to each year for each product.

# Write your MySQL query statement below
with first_y as (
    select *,
    row_number() over (partition by product_id order by year asc) as c
    from sales
    
),

prod_y as (
    select product_id, year
    from first_y
    where c = 1
)

select s.product_id, s.year as first_year, s.quantity, s.price
from sales s inner join prod_y y on s.product_id = y.product_id and s.year = y.year

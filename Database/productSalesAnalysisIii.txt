    - The outer query then joins this inner query back to the original `sales` table. The join uses `product_id` and `year` to match the first sales corresponding to each product, allowing us to retrieve the additional columns – `quantity` and `price` – for the first sale.

   - **Time Complexity**:

select s.product_id, s.year as first_year, s.quantity, s.price
from sales s
join (
    select product_id, min(year) as year
    from sales
    group by product_id
) f using (product_id, year)
